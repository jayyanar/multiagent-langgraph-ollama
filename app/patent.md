Personality-Driven Consequence Reasoning (PDCR): A Five-Layer Deliberative Architecture for Consequence-Aware Household Robotics

Abstract:
Carrying out multi-step tasks in unstructured environments has been significantly improved with recent Vision-Language-Action (VLA) architectures-based models for household robots. Existing systems still tend to be primarily reactive, maximizing short-horizon task success but without directly reasoning about downstream safety, social or user-dependent consequences. This limitation, which we call consequence blindness, induces a gap between task competence and real-world acceptability in human-centric environments. We propose the Personality-Driven Consequence Reasoning (PDCR), a deliberative framework where multi-dimensional consequence evaluation is directly integrated into the robot decision loop. We propose to split control in two: a high-speed reactive path and a consequence-aware evolution module at the high level to evaluate candidate trajectories according to physical risk, safety, social acceptability and task-functional dimensions. Additionally, this framework integrates an uncertainty-aware user model to update soft behavioral preferences while maintaining fixed hard safety constraints. We instantiate the proposed architecture as a modular embodied stack and evaluate it on simulated, immersive household tasks and safety-critical planning. The results obtained after using the PDCR algorithm show that the actual state of the system is maintained along with the uncertainty estimation on personality probabilities are minimized as compared to existing reactive systems, which makes it executed in a safe environment.
